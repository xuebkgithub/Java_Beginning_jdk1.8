# 基本数据类型
## 数值型
### 整数型
* byte: 占用1个字节=8bit 表述范围 (-128 到 127)
* short: 占用2个字节=16bit 表述范围 (-2的15次方 到 2的15次方-1)
* int: 占用4个字节=32bit 表述范围 (-2的31次方 到 2的31次方-1)
* long: 占用8个字节=64bit 表述范围 (-2的63次方 到 2的63次方-1)
